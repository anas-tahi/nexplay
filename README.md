## 🕹️ Games Included

| Game | Description |
|---|---|
| 🐍 Snake | Classic snake — grow without hitting yourself |
| 🧠 Memory | Flip cards and match pairs |
| ❌⭕ Tic-Tac-Toe | Classic 3x3 strategy game |
| 🏓 Pong | Two-player paddle battle |
| 🧱 Breakout | Destroy all blocks with your ball |

## ✨ Features

- 🔐 User auth — register, login, or play as guest
- 🏆 Global leaderboards per game
- 📊 Personal score history
- 📱 Responsive design

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React |
| Backend | Node.js + Express |
| Database | MongoDB |
| Auth | JWT |

## 🚀 Run Locally

```bash
git clone https://github.com/anas-tahi/nexplay
cd nexplay

# Backend
cd backend && npm install && npm start

# Frontend
cd frontend && npm install && npm run dev
```
